💡 HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ADMIN CREATES ANNOUNCEMENT:
────────────────────────────
1. Admin clicks "Create Announcement"
2. Fills title, type, message
3. Clicks "Publish to All Employees"
4. Updates ALL active employees' Announcement field
5. Everyone (admin + employees) sees it immediately

EMPLOYEE SEES ANNOUNCEMENT:
───────────────────────────
1. Navigates to any page
2. Banner shows at top
3. Can read the message
4. Can click ✕ to dismiss

EMPLOYEE DISMISSES:
───────────────────
1. Click ✕ button
2. Saves to localStorage with timestamp
3. Banner hides
4. After 24 hours, reappears automatically
5. Or reappears immediately if admin creates new one

ADMIN EDITS/DELETES:
────────────────────
1. Admin clicks ✏️ (edit) or ✕ (delete)
2. Updates all active employees
3. Change reflects for everyone instantly
4. Employees' dismissals are cleared on new announcement
